📊 Restaurant Sales Analysis using SQL
📌 Project Overview

This project analyzes restaurant sales data using SQL to identify business performance, customer behavior, product popularity, and revenue trends. The objective is to derive actionable business insights that help improve operational efficiency and profitability.

🎯 Business Problem

The restaurant management wants to understand:

Which food items generate maximum revenue

Customer purchasing behavior

Peak sales periods

Preferred payment methods

Opportunities to increase revenue

SQL analysis was performed to transform raw sales data into meaningful business insights.

🛠️ Tools & Technologies

SQL (MySQL)

Data Cleaning

Aggregations & Grouping

Window Functions

Business Analysis

📂 Dataset Information

The dataset contains restaurant transaction records including:

Order ID

Order Date

Food Item

Category

Quantity

Price

Payment Method

Customer Name

📈 Key SQL Analysis Performed

✅ Data Cleaning
✅ Revenue Analysis
✅ Product Performance Analysis
✅ Customer Analysis
✅ Payment Method Analysis
✅ Time-Based Sales Analysis
✅ Average Order Value Calculation

📊 Key Business Insights

A small number of food items contribute to the majority of total revenue.

Main course items generate the highest sales.

Digital payments are preferred by customers.

Sales peak during specific weekdays.

Loyal customers contribute significantly to revenue.

💡 Business Recommendations

Increase inventory for top-selling products

Introduce combo meal offers

Implement loyalty programs

Optimize staffing during peak hours

Promote low-performing menu items

🚀 Project Outcome

This project demonstrates how SQL can be used to solve real-world business problems by converting raw transactional data into strategic insights for decision-making.
